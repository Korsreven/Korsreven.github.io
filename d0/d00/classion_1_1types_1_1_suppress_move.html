Detailed Description

template<typename T>
class ion::types::SuppressMove< T >

A class representing a member value that has suppressed copies and moves.

Tag class members as SuppressMove to disable copying and moving of its value. An object containing one or more tagged members can still be copied and moved normally. A member tagged as SuppressMove will be default constructed when the copy or move constructor is called
