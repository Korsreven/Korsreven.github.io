Detailed Description

template<typename T>
class ion::types::SuppressCopy< T >

A class representing a member value that has suppressed copies.

Tag class members as SuppressCopy to disable copying of its value. An object containing one or more tagged members can still be copied normally. A member tagged as SuppressCopy will be default constructed when the copy constructor is called
