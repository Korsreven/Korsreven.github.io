Detailed Description

A class representing a tree structure with object, property and argument nodes.

The tree and nodes are created during compilation of one or more translation units. A script tree needs to be validated against a validation scheme for it to be considered correct. A script tree can be validated then serialized, to later be deserialized without the need to be validated again
